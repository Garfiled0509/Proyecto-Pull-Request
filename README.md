## Descripció de la Pràctica

Aquesta pràctica serveix per familiaritzar-se amb les ordres bàsiques de control de versions de Git i la plataforma GitHub. Els objectius principals han estat:

* Clonar un repositori.
* Crear noves branques (`feature/ShoppingCard` i `feature/ShowProduct`).
* Realitzar 'commits' de canvis a diferents branques.
* Publicar les branques al repositori remot.
* Crear, gestionar i fusionar Pull Requests (PRs).
* Resoldre un conflicte de fusió (merge conflict) manualment durant una PR.
* Mantenir un historial de 'commits' net i organitzat.
